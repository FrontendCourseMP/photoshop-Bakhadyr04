# Цифровое представление изображения

Веб-приложение на React + Vite для загрузки, просмотра и сохранения изображений в форматах `PNG`, `JPG` и `GB7`.

## Ссылка на приложение
https://photoshop-bakhadyr04.onrender.com/


## Возможности

- загрузка файлов через picker и drag-and-drop
- декодирование и кодирование формата GrayBit-7
- адаптивный предпросмотр через HTML5 Canvas
- строка состояния с размерами, глубиной цвета, размером файла и форматом
- экспорт изображения в `PNG`, `JPG` и `GB7`

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

Основной код находится в `src/`, включая:

- `components/` для UI
- `hooks/useImageData.js` для загрузки и сохранения
- `utils/gb7-decoder.js` и `utils/gb7-encoder.js` для формата GB7
